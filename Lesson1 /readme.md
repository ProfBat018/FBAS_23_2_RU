## Программный проект vs проект

- Проект — это временное начинание, имеющее начало и конец, целью которого является создание уникального продукта, услуги или результата.
- Программный проект — это проект, целью которого является создание программного обеспечения.

Внутри проекта могут быть другие проекты, которые могут быть как программными, так и непограммными. Например, проект по созданию нового автомобиля может включать в себя проект по созданию программного обеспечения для управления автомобилем, а также проект по созданию нового двигателя.

## Диаграмма Ганта

![Диаграмма Ганта](https://image.mindbox.ru/storage/9426c370-9ed0-47a2-9704-c285874e14f0.png)

Данная диаграмма нужна для отслеживания и визуализации задач в контексте командной разарботки проекта. Она улучшает разработку путем планирования задач и их сроков, а также позволяет отслеживать прогресс выполнения задач. Диаграмма Ганта позволяет увидеть, какие задачи выполняются в данный момент, какие задачи уже выполнены, а какие еще предстоит выполнить. Она также позволяет увидеть, какие задачи зависят друг от друга и как они связаны между собой. Одной из вариаций данной диаграммы является `Kanban`.
